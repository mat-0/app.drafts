# app.drafts

A repo containing scripts and snippets of useful stuff used in the [Drafts app](https://getdrafts.com/) where text starts.

## Contents

Each file is prefixed with a type to indicate its intended use:

- `actions -` Assigned as actions in the actions menu as typically used at end of process.
- `keys -` Assigned as keys in the shortcut keyboard for quick access as typically during document creation or mid-process.
- `templates -` Assigned as templates for creating new drafts using the new-draft template system.
