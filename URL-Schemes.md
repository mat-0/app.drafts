# URL Schemes for Drafts

This document provides a list of useful URL schemes for the [Drafts app](https://getdrafts.com/). These URL schemes can be used to automate actions, create new drafts, and interact with the Drafts app programmatically.

## Common URL Schemes

### Signal

- `sgnl://` - Open the Signal app.

### Telegram

- `tg://msg?text=[[draft]]` - Share the current draft text via Telegram.

### WhatsApp

- `whatsapp://send?text=[[draft]]` - Share the current draft text via WhatsApp.

### ISBN Search

- `https://isbnsearch.org/search?s=[[selection]]`

### Amazon Search

- `http://www.amazon.co.uk/s/field-keywords=[[selection]]`

### Arc Search

- `arcmobile2://search?query=[[selection]]`

### DuckDuckGo Search

- `https://duckduckgo.com/?q=[[selection]]`
- `ddg://search?query=[[selection]]`

### Bing Search

- `bing://search?query=[[selection]]`
